# Calculus Examples

examples demonstrate differentiation, integration, and their interplay.

| [Mean Value Theorem](mean_value_theorem.md) | Finding MVT points via roots |
| [Differentiation](differentiation.md) | `diff` and the fundamental theorem |
| [Snell's law](snells_law.md) | Geometric optics via optimization |
| [Bird flight optimization](bird_flight_optimization.md) | Energy minimization with calculus |
