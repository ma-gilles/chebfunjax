# Temporary / Miscellaneous Examples

A collection of examples from various Chebfun categories including
fractional calculus, ODEs on unbounded domains, optimal control,
and integro-differential equations.

| Example | Description |
|---------|-------------|
| [Binous-Shaikh-Bellagi](binous_shaikh_bellagi.md) | Nonlinear BVPs, heat equation, and convection-diffusion from BSB |
| [Chebfun Composition](chebfun_composition.md) | Function composition: g(f) for 1D and 2D Chebyshev functions |
| [Compacting Colloids](compacting_colloids.md) | AJR sedimentation PDE in a centrifuge; stiff time-stepping |
| [Fractional Calculus](frac_calc.md) | Riemann-Liouville D^α for α=0.1,...,1; half-derivative of x |
| [Fractional Calculus 2](frac_calc2.md) | SH coefficient formulas for fractional integrals |
| [Integro-Differential](integro_diff_t.md) | Time-dependent integro-differential PDE with nonlocal coupling |
| [Optimal Car](optim_car.md) | Bang-bang optimal control: car position and speed on [0,2] |
| [Piecewise Linop Demo](piecewise_linop_demo.md) | Piecewise ODE with sign(x) coefficient and jump continuity |
| [Taylor's Theorem](taylors_theorem.md) | Chebyshev analogue: convergence in Bernstein ellipses |
| [Unbounded ODE](unbounded_ode.md) | 0.1u''+u'+u=0 on [0,∞); domain truncation and piecewise variant |
| [Van der Pol Orbit](vdp_orbit.md) | Limit cycle of u''−μ(1−u²)u'+u=0 for various μ |
