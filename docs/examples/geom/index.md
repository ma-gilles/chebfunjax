# Geometry Examples

Chebfunjax excels at computing with parametric curves and surfaces: arclengths,
areas, perimeters, and more — all computed to near machine precision.

|---------|-------------|
| [Lissajous Curves](lissajous.md) | x=sin(mt), y=sin(nt+δπ): grid of Lissajous figures |

| [Rose Curves](rose_curves.md) | r=cos(m/n·θ): 6×6 grid of polar rose curves |
| [Area and Centroid](area_centroid.md) | Epicycloid area and centroid via Green's theorem |
| [Two Circles](two_circles.md) | Overlap area of two circles: Povey's exact solution |
| [Ellipses Rolling](ellipses_rolling.md) | ODE for an ellipse rolling on an ellipse |
| [Rounding Corners](rounding_corners.md) | W-shape convolved with a tent kernel |
| [Curves Distance](curves_distance.md) | Minimum distance between two random curves |
| [Constant Width](constant_width.md) | Rabinowitz degree-8 polynomial zero set (curve of constant width) |
| [Sinai Billiards](sinai_billiards.md) | Photon bouncing off circular mirrors |
| [Procrustes](procrustes.md) | Translate, scale, rotate shape alignment |
| [Dual Curves](dual_curves.md) | Point-line duality, dual of heart and ellipse |
