# Linear ODE Examples

Chebfun solves linear boundary value problems (BVPs) and eigenvalue problems
via spectral collocation. These examples cover the classical linear ODEs.

| Example | Description |
|---------|-------------|
| [Boundary layer](boundary_layer.md) | Advection-diffusion with small ε |
| [Airy equation](airy_equation.md) | $u'' = xu$ — Airy functions |
| [Bessel BVP](bessel_bvp.md) | Bessel's equation as a BVP |
| [Poisson equation](poisson_equation.md) | $-u'' = f$ with Dirichlet BC |
| [Linear IVP](linear_ivp_cosine.md) | $u' = \cos(t)$, $u(0)=0$ |
| [Wiki ODEs](wiki_odes.md) | Classical ODEs from Wikipedia |
