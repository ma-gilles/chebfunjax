# Fun/Miscellaneous Examples

A collection of examples showcasing the fun side of chebfunjax: fractals,
curves, cryptography, and creative mathematical art.

| Example | Description |
|---------|-------------|
| [Audible Chebfuns](audible_chebfuns.md) | Chebyshev coefficient spectra as audio frequency representations |
| [Birthday Cards](birthday.md) | Piecewise-linear complex paths for writing messages |
| [Birthday Odds](birthday_odds.md) | The birthday paradox: 23 people needed for 50% shared-birthday chance |
| [Contour Fun](contour_fun.md) | Implicit curves from contour lines: circles, lemniscates, Cassini ovals |
| [Encryption](encryption.md) | Message encryption via complex rotations and scaling |
| [Hello World](hello_world.md) | "Hello" displayed via 2D Chebyshev function zero sets |
| [Hilbert Surface Chebsnake](hilbert_surface_chebsnake.md) | Hilbert space-filling curve and Chebsnake traversal of Chebyshev grids |
| [Koch Snowflake](koch_snowflake.md) | Koch fractal and Hausdorff dimension log(4)/log(3) |
| [MathJax](mathjax.md) | Key Chebyshev formulae and orthogonality matrix visualization |
| [Piecewise Linear](piecewise_linear.md) | Labyrinth path, Chebyshev interpolant, and Lebesgue constant comparison |
| [Scribble 2](scribble2.md) | Zero sets of 2D algebraic and Chebyshev series equations |
| [Soccer Ball](soccer_ball.md) | Truncated icosahedron: wireframe, stereographic projection, face composition |
| [Valentine's Day](valentines_day.md) | Heart curves: areas via Green's theorem |
| [Valentine's Day 2](valentines_day2.md) | 3D heart surface, beating animation, and rose of hearts |

| [Xmas Card](xmas_card.md) | "Kling Gloeckchen" melody as piecewise-constant chebfun |
