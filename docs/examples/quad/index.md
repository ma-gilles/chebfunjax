# Quadrature Examples

Chebfun integrates smooth functions by analytically integrating the
Chebyshev expansion. These examples explore Gauss, Clenshaw-Curtis,
and related quadrature rules.

|---------|-------------|
| [Gauss vs. Clenshaw-Curtis](gauss_vs_clenshaw_curtis.md) | Convergence comparison |
| [Gauss quadrature](gauss_quadrature.md) | Nodes and weights via Golub-Welsch |
| [Clenshaw-Curtis](clenshaw_curtis.md) | Integration via FFT |
| [Convergence rates](convergence_rates.md) | Algebraic convergence for non-smooth functions |
| [Tricky integrals](tricky_integrals.md) | Handling singularities and oscillations |
