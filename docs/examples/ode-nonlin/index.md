# Nonlinear ODE Examples

Chebfun solves nonlinear BVPs via Newton's method (the `solve` method
with automatic differentiation). These examples show problems from
physics and engineering.

| Example | Description |
|---------|-------------|
| [Carrier equation](carrier_equation.md) | Multiple solutions via different initial guesses |
| [Allen-Cahn](allen_cahn.md) | Reaction-diffusion with sharp interfaces |
| [Logistic equation](logistic_equation.md) | Population dynamics IVP |
| [Pendulum equation](pendulum_equation.md) | Nonlinear oscillator |
| [Exact solutions](exact_solutions_bender_orszag.md) | Bender-Orszag benchmark problems |
