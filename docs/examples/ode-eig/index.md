# ODE Eigenvalue Examples

Differential operators have spectra — eigenvalues and eigenfunctions.
Chebfun computes these spectra via the `eigs` method.

| Example | Description |
|---------|-------------|
| [Harmonic oscillator](harmonic_oscillator.md) | Quantum harmonic oscillator eigenstates |
| [Double well](double_well.md) | Tunnel splitting in a symmetric well |
| [Laplacian eigenvalues](laplacian_eigenvalues.md) | $-u'' = \lambda u$ on an interval |
| [Sturm-Liouville](sturm_liouville.md) | General S-L eigenvalue problem |
